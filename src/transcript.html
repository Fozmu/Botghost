Ticket Transcripts

Are you looking for a free ticket transcript? We have a step-by-step guide listed below:

  1. Head over to Cookie API
  2. Creat an account and head over to Overview. Once there copy the Api Key.
  3. Go back to Botghost and add an API request.

  4. Set the methode as Post and as the url enter https://api.cookie-api.com/api/transcript
  5. In the url Paragrams add channel_id as the key and enter a valid channel id in the value section.
    (This can be replaced by a variable after step 10.)
  6. Head over to the HTTP Headers and add Authorization as your key. In the value section enter the API Key from before.
  7. Head over to the Request Body. Add bot_token as the key.
  8. For the value we will need the bot token. {TOKEN_SECRET} wont work. we will need to use our real bot token.
    (Bot token can be reset in the Discord developer page under the bot section.)



  9. Test the request if you get a code 400 check for any typos and make sure you dont have any variables.
    If you get a code 200 everything worked.
  10. copy the variable for the url that you received.
  11. Replace all ids with variables so you can use it in your system.

You should be all done, if you have any questions ask a CS member.
